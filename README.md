# 🚀 claude-plugins-cli

> code formatter with opinionated defaults

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g claude-plugins-cli
claude-plugins-cli init my-project
```

## 🛠️ Stack

Built with leaflet-Nextjs + analyze
